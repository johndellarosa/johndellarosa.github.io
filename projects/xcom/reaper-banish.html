XCOM 2 Banish Simulation

Return to Menu

Simulates the outcome of using the Reaper's Banish ability. To use, set the input values to match the test scenario then press the "Run Simulation" button. Preset values for each tier can be applied by clicking on the weapon names.

To do: incorporate dodging.

The first graph (histogram) tells what percentage of trials ended up requiring that number. The second graph shows the cumulative distribution function (CDF), which shows what fraction of trials where the enemy had at most that amount of HP remaining.

For example, find the x-value for the pointer corresponding to y=0.25. 25% of the time, the enemy would have at most whatever that x-value is as their remaining HP.
