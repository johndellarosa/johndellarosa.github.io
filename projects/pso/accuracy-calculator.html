Phantasy Star Online Accuracy Calculator

Return to Menu

Calculates the percent chance of an attack dealing damage to an enemy. Currently accuracy drop due to distance with ranged weapons is not supported. Single player values are displayed.

  1. Letter refers to type of attack: N=Normal, H=Heavy, S=Special
  2. Number refers to which hit in the combo it is
  3. Values above 100 mean 100% to hit. Values below 0 mean 0% chance to hit.
  4. Set Distance parameter to a non-zero value only if you are using non-ranger class with a ranged weapon. Leave as 0 if using melee weapons and/or if you are a ranger.
  5. Handgun max range is 170 units. Mech gun max range is 85 units. Slicer max range is 95 units.
